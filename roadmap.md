# 🪨 Hajar Programming Language Roadmap

This roadmap describes the planned development of the Hajar Programming Language.

> The roadmap may change as the project evolves.

---

# v0.1.0-alpha (Current)

## Interpreter

- [x] Interactive REPL
- [x] Execute `.hjr` files
- [x] `show()` command
- [x] Comment support (`#`)
- [x] Basic error handling

---

# v0.2.0-alpha

## Language Basics

- [ ] Variables
- [ ] Numbers
- [ ] Strings
- [ ] Basic operators

Example:

```hjr
name = "Abdullah"
show(name)
```
