# 🪨 Hajar Programming Language

Hajar is an experimental open-source programming language written in Python.

The project is currently in a very early stage (**v0.1.0-alpha**) and is intended for learning programming language design, interpreters, and compiler development.

The current version is a simple interpreter written entirely in Python.

## Features

- Interactive REPL
- Execute `.hjr` source files
- `show("text")` command
- Comment support using `#`
- Simple error handling

## Example

```hjr
# My first Hajar program

show("Hello, World!")
show("Welcome to Hajar Programming Language!")
```

## Running

Interactive mode:

```bash
python hajar.py
```

Run a file:

```bash
python hajar.py hello.hjr
```

## Roadmap

Current (0.1.0-alpha)

- Basic interpreter
- REPL
- Source file execution
- show() command

Future versions

- Variables
- Functions
- Control flow
- Lexer
- Parser
- Standard library
- Faster execution
- Native compiler

## Goals

The long-term vision of Hajar is to become a programming language that is:

- Easy to learn
- Powerful
- Fast
- Open source
- Community-driven
- Suitable for different kinds of software development
