# 🪨 لغة البرمجة حجر

🇸🇦 العربية | 🇺🇸 [English](README.md)

حجر هي لغة برمجة مفتوحة المصدر وتجريبية، مكتوبة بالكامل بلغة Python.

لا يزال المشروع في مراحله الأولى (**0.1.0-alpha**) ويهدف إلى تعلم تصميم لغات البرمجة وبناء المفسرات والمترجمات.

الإصدار الحالي عبارة عن مفسر بسيط مكتوب بلغة بايثون.

## المميزات

- وضع تفاعلي (REPL)
- تشغيل ملفات `.hjr`
- الأمر `show("النص")`
- دعم التعليقات باستخدام `#`
- معالجة بسيطة للأخطاء

## مثال

```hjr
# أول برنامج بلغة حجر

show("مرحباً بالعالم")
show("أهلاً بك في حجر")
```

## التشغيل

الوضع التفاعلي

```bash
python hajar.py
```

تشغيل ملف

```bash
python hajar.py hello.hjr
```
